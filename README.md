# WishLocker Inline Bot

Инлайн‑бот для Telegram, который помогает вести игровой **счёт желаний** между двумя людьми.

Бот показывает текущее количество баллов и накопленных «желаний» у каждого участника и позволяет:

- **Начислять / списывать очки** кнопками.
- **Одновременно добавлять очки обоим** участникам.
- **Конвертировать 100 очков в одно «желание»**.
- **Тратить желания** с красивым обновлением сообщения.

Работает в **inline‑режиме**: достаточно написать `@имя_бота` в любом чате.

---

## Стек

- **Python 3**
- **[aiogram](https://docs.aiogram.dev/)** — фреймворк для Telegram‑ботов

---

## Возможности

- **Инлайн‑режим**: бот вызывается через `@bot_username` без отдельного чата.
- **Живое обновление сообщения**: счёт и желания обновляются в уже отправленном сообщении.
- **Кнопки управления**:
  - `+1` / `-1` для каждого участника
  - `+1 обоим` (⭐)
  - «Потратить желание» для каждого
- **Гибкий ввод**: можно написать число после имени бота (например, `@bot_username 25`), чтобы разово добавить это количество очков.

## Установка и запуск

1. **Клонировать репозиторий**:

   ```bash
   git clone https://github.com/BUka228/WishLocker_Inline_Bot.git
   cd WishLocker_Inline_Bot
   ```

2. **Создать и активировать виртуальное окружение (рекомендуется)**:

   ```bash
   python3 -m venv venv
   source venv/bin/activate  # macOS / Linux
   # для Windows: venv\Scripts\activate
   ```

3. **Установить зависимости** (если добавлен `requirements.txt`):

   ```bash
   pip install -r requirements.txt
   ```

4. **Настроить токен бота**:

   В файле `bot.py` используется константа `TOKEN`. Рекомендуется **хранить токен в переменной окружения**, а не в коде. Например:

   ```python
   import os

   TOKEN = os.getenv("BOT_TOKEN")
   ```

   А затем перед запуском:

   ```bash
   export BOT_TOKEN="<ваш_токен_от_BotFather>"
   ```

5. **Запустить бота**:

   ```bash
   python bot.py
   ```

## Как пользоваться

1. Добавьте бота в Telegram и запустите его (`/start` при необходимости).
2. В любом чате (личном, групповом, с самим собой) начните вводить:

   ```text
   @имя_бота
   ```

3. Выберите карточку «Показать текущий счёт» и отправьте её в чат.
4. Используйте кнопки под сообщением, чтобы:

   - Добавлять или отнимать очки у каждого участника.
   - Начислять очки сразу обоим.
   - Тратить накопленные желания.

5. Чтобы добавить произвольное количество очков, напишите число после имени бота:

   ```text
   @имя_бота 50
   ```

   Появится вариант начислить именно это количество выбранному участнику.
